# Детектирование товаров на полках супермаркетов (SKU-110K)

Сравнительный анализ пяти современных моделей детектирования объектов
(YOLOv8n, YOLO11n, SSDLite, Faster R-CNN, RT-DETR-L) в задаче обнаружения
товаров на изображениях торговых полок — плотноупакованных сценах со
~147 объектами на изображение в среднем.

Проект выполнен в рамках учебной практики (МТУСИ, кафедра МКиИТ, 2026).

## Результаты

| Модель | mAP@50 | mAP@50-95 | F1 | Параметры, млн | Инференс, мс |
|---|---|---|---|---|---|
| YOLOv8n | **0.850** | **0.498** | 0.827 | 3.01 | 5.3 |
| YOLO11n | 0.846 | 0.495 | 0.824 | 2.59 | 13.0 |
| RT-DETR-L | 0.797 | 0.449 | 0.799 | 32.81 | 17.2 |
| Faster R-CNN | 0.436 | 0.155 | 0.511 | 18.93 | 24.6 |
| SSDLite | 0.116 | 0.021 | 0.251 | 2.21 | 12.2 |

Тестовая выборка: 300 изображений, 43 875 объектов. Графики обучения,
сравнительные диаграммы и примеры предсказаний — в `results/plots/`.

## Данные

Подвыборка [SKU-110K](https://github.com/eg4000/SKU110K_CVPR19)
(Goldman et al., CVPR 2019): 1500 train / 300 val / 300 test изображений,
приведённых к 640 px, разметка в формате YOLO. Скачивается и готовится
автоматически (стриминг с HuggingFace, полные 13 ГБ не требуются).

## Структура проекта

```
configs/            конфигурации моделей и экспериментов (yaml)
data/raw|processed  данные
src/dataset/        подготовка данных, PyTorch Dataset
src/models/         фабрика torchvision-моделей (адаптация под плотные сцены)
src/training/       циклы обучения (ultralytics / torchvision)
src/evaluation/     метрики (mAP, P/R/F1), оценка на тесте
src/utils/          конфиг, логирование, визуализация
notebooks/          исследовательский анализ данных (EDA)
results/            логи обучения, метрики, графики
main.py             точка входа
```

## Запуск

```bash
pip install torch torchvision --index-url https://download.pytorch.org/whl/cu126
pip install -r requirements.txt

python main.py --stage prepare        # скачивание и подготовка данных
python main.py --stage all            # обучение и оценка всех 5 моделей
python main.py --stage experiments    # эксперименты с гиперпараметрами
python main.py --stage plots          # графики и сравнительная таблица
python main.py --stage train --model yolov8n   # одна модель
```

## Воспроизводимость

Фиксированный seed (42), детерминированный режим, все гиперпараметры — в
`configs/default.yaml`, фактические конфигурации и логи каждого запуска — в
`results/logs/<run>/`. Обучение выполнялось на одном GPU NVIDIA RTX 3050
Laptop (4 ГБ): лимит детекций на изображение поднят до 300 (плотные сцены),
размеры батча и разрешения подобраны под бюджет видеопамяти, использована
смешанная точность (AMP).
